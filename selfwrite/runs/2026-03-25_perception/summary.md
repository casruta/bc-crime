## Key Learnings

1. **Corrective tone requires data authority upfront.** The biggest corrective-clarity gain (+1 to 8) came from asserting "police-reported, sentence-weighted, statistically significant (p < 0.05)" in the opening. Policy-makers need to know the data source is authoritative before they'll accept the corrective frame.

2. **Inline correctives + summary table > either alone.** The inline format lets a reader get the action while reading the mechanism. The table lets a reader skip to assignments. Both serve the policy-maker audience differently.

3. **The 30%/30% parallel was the highest-leverage data point.** Linking "property crime clears below 30%" to "only 30% have great confidence" made the feedback loop numerically visible. Policy-makers respond to patterns in numbers, not assertions about trust.

4. **The closing bookend earned +0.35.** "The question is not whether the public can be convinced... it is whether the institutions can earn the credibility to be believed" reframes the problem from public ignorance to institutional accountability — exactly what a corrective tone demands.
