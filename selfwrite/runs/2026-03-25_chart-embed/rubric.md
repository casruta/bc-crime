# Chart Embedding — Scoring Rubric

## Dimensions

### 1. Completeness (Weight: 0.25)
- **1-2:** Most charts missing
- **5-6:** All charts present but some misplaced or missing captions
- **9-10:** All 42 PNGs embedded in correct sections with accurate captions; HTML map linked

### 2. Visual Integration (Weight: 0.25)
- **1-2:** Charts dumped at section end with no relationship to prose
- **5-6:** Charts near relevant prose but placement feels mechanical
- **9-10:** Each chart appears immediately after the prose that references it; charts and text form a unified narrative

### 3. Caption Quality (Weight: 0.20)
- **1-2:** No captions or generic "Figure 1"
- **5-6:** Captions describe what the chart shows but not the takeaway
- **9-10:** Each caption states the key finding visible in the chart (point-first); figure numbered

### 4. Scannability (Weight: 0.15)
- **1-2:** Wall of images, no grouping or visual hierarchy
- **5-6:** Some grouping but reader can't quickly find a specific chart
- **9-10:** Clear section headers, logical grouping, reader can jump to any chart via TOC

### 5. Narrative Flow (Weight: 0.15)
- **1-2:** Charts interrupt the reading experience
- **5-6:** Charts fit but prose doesn't reference them
- **9-10:** Prose references charts naturally; charts reinforce the argument
